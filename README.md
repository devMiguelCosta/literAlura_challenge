📘 LiteAlura - Catálogo de Livros
Desafio desenvolvido durante a formação Oracle Next Education + Alura. A aplicação consome a API Gutendex para gerenciar um catálogo de livros e autores, persistindo os dados em um banco de dados relacional.


🚀 Funcionalidades
Busca Inteligente: Pesquisa livros pelo título via API e salva automaticamente no DB.

Listagem Completa: Visualização de todos os livros e autores registrados.

Filtro Temporal: Listar autores vivos em um determinado ano.

Filtro por Idioma: Busca segmentada por código de língua (PT, EN, FR, etc).

Validação de Dados: Sistema que impede a duplicidade de registros e trata entradas inválidas.

🛠️ Tecnologias & Arquitetura
O projeto utiliza uma estrutura modular seguindo as melhores práticas de Clean Code:

Java 17 / 21 (LTS)

Spring Boot 3.x

Spring Data JPA (Persistência e queries JPQL)

PostgreSQL (Armazenamento robusto)

Json (Desserialização de JSON complexos)
